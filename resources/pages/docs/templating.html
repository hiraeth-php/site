{% extends '@layouts/docs.html' %} {% set title = 'Templating' %} {% block content %} {% markdown %} ## Register Template Paths In the file `config/packages/templates.jin` you can modify template paths accordingly: ```toml [templates] paths = { "pages": "resources/pages", "layouts": "resources/layouts" } ``` The key is used as an alias when loading the template so that `@pages/home.html` would be expanced to `resources/pages/home.html`.
