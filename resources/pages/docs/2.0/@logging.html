{% extends '@layouts/docs-2.0.html' %} {% set title = 'Logging' %} {% block content %} {% markdown %} {: .note} This document assumes you've installed `hiraeth/bootstrap`{: .package} or `hiraeth/monolog`{: .package}. If you haven't your mileage may vary. See the [installation docs](./installation) for more information. Logging in Hiraeth is performed using [Monolog](https://github.com/Seldaek/monolog) which adhere's to the [PSR-3](https://www.php-fig.org/psr/psr-3/) FIG standard. You can receive an instance of the logger by typehinting `Psr\Log\LoggerInterface`{: .interface}. {% endmarkdown %} {% endblock %}
